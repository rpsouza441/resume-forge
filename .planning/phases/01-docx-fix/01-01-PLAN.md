   - Based on: Normal
   - Used by: createSectionHeading()

3. **ResumeBullet** (styleId: "ResumeBullet", name: "Resume Bullet")
   - Font: Arial, 10pt (20 half-points)
   - Indentation: Left 0.5", Hanging 0.25"
   - Used by: createBulletItem()

4. **ResumeJobPeriod** (styleId: "ResumeJobPeriod", name: "Resume Job Period")
   - Font: Arial, 9pt (18 half-points), Italic
   - Based on: Normal
   - Used by: createExperienceEntry() for period line

Implementation pattern:
```java
private XWPFStyles styles;

private void createNamedStyles(XWPFDocument document) {
    this.styles = document.createStyles();
    
    // ResumeName style
    XWPFStyle nameStyle = styles.createStyle(STYLE_TYPE.PARAGRAPH, "ResumeName");
    nameStyle.setName("Resume Name");
    nameStyle.setFontSize(32);
    nameStyle.setFontFamily("Arial");
    nameStyle.setBold(true);
    
    // ResumeSectionHeading style
    XWPFStyle sectionStyle = styles.createStyle(STYLE_TYPE.PARAGRAPH, "ResumeSectionHeading");
    sectionStyle.setName("Resume Section Heading");
    sectionStyle.setFontSize(24);
    sectionStyle.setFontFamily("Arial");
    sectionStyle.setBold(true);
    
    // ResumeBullet style
    XWPFStyle bulletStyle = styles.createStyle(STYLE_TYPE.PARAGRAPH, "ResumeBullet");
    bulletStyle.setName("Resume Bullet");
    bulletStyle.setFontSize(20);
    bulletStyle.setFontFamily("Arial");
    
    // ResumeJobPeriod style
    XWPFStyle periodStyle = styles.createStyle(STYLE_TYPE.PARAGRAPH, "ResumeJobPeriod");
    periodStyle.setName("Resume Job Period");
    periodStyle.setFontSize(18);
    periodStyle.setFontFamily("Arial");
    periodStyle.setItalic(true);
}
```

Call `createNamedStyles(document)` in `createDocument()` after `this.numbering = document.createNumbering()`.
